import pandas as pd
from sklearn.model_selection import train_test_split
from sklearn.naive_bayes import GaussianNB
from sklearn.metrics import confusion_matrix, accuracy_score, precision_score, recall_score

# -------------------------------
# Load Dataset
# -------------------------------
df = pd.read_csv('Iris Dataset Prac (1,3.2,6,10).csv')

# -------------------------------
# Feature Selection
# -------------------------------
# Independent variables
X = df[['SepalLengthCm', 'SepalWidthCm', 'PetalLengthCm', 'PetalWidthCm']]

# Target variable
y = df['Species']

# -------------------------------
# Train-Test Split
# -------------------------------
X_train, X_test, y_train, y_test = train_test_split(
    X, y, test_size=0.20, random_state=42)

# -------------------------------
# Model Creation and Training
# -------------------------------
model = GaussianNB()
model.fit(X_train, y_train)

# -------------------------------
# Prediction
# -------------------------------
y_pred = model.predict(X_test)

# -------------------------------
# Confusion Matrix
# -------------------------------
cm = confusion_matrix(y_test, y_pred)
print("Confusion Matrix:\n", cm)

# -------------------------------
# Evaluation Metrics
# -------------------------------
accuracy = accuracy_score(y_test, y_pred)
precision = precision_score(y_test, y_pred, average='macro')
recall = recall_score(y_test, y_pred, average='macro')
error_rate = 1 - accuracy

print(f"\nAccuracy: {accuracy:.2f}")
print(f"Error Rate: {error_rate:.2f}")
print(f"Precision: {precision:.2f}")
print(f"Recall: {recall:.2f}")

# -------------------------------
# TP, FP, TN, FN for each class
# -------------------------------
print("\n--- TP, FP, TN, FN for each class ---")
for i in range(len(cm)):
    TP = cm[i][i]
    FN = sum(cm[i]) - TP
    FP = sum(cm[:, i]) - TP
    TN = cm.sum() - (TP + FP + FN)

    print(f"\nClass {i}:")
    print(f"TP={TP}, FP={FP}, FN={FN}, TN={TN}")

Introduction

Naïve Bayes is a supervised machine learning algorithm used for classification tasks. It is based on Bayes’ Theorem and assumes that features are independent of each other. In this practical, the Naïve Bayes classifier is applied on the Iris dataset to classify flower species.

Naïve Bayes Concept

Naïve Bayes is based on:

P(C∣X)=P(X∣C)⋅P(C)  /
P(X)

	​
Where:

P(C∣X) → Posterior probability
P(X∣C) → Likelihood
P(C) → Prior probability
P(X) → Evidence
1. Dataset Description

The Iris dataset contains:

Sepal Length
Sepal Width
Petal Length
Petal Width
Species (target variable)

There are three classes:

Setosa
Versicolor
Virginica
2. Model Used: Gaussian Naïve Bayes

GaussianNB is used because:

Data is continuous
Assumes features follow normal distribution
3. Training the Model

The dataset is split into training and testing sets.
The model learns probabilities of each class based on feature values.

4. Prediction

The model predicts the class label for test data by calculating posterior probabilities and selecting the class with the highest probability.

5. Confusion Matrix

For multi-class classification, confusion matrix is a square matrix showing correct and incorrect predictions.

Each row represents actual values, and each column represents predicted values.

6. Evaluation Metrics
Accuracy
Accuracy=Correct Predictions   /
Total Predictions

	​

Error Rate
Error Rate=1−Accuracy
Precision

Measures correctness of positive predictions.

Recall

Measures ability to identify correct classes.

Macro averaging is used to calculate average metrics across all classes.

7. TP, FP, TN, FN in Multi-Class

For each class:

TP: Correct predictions of that class
FP: Other classes wrongly predicted as this class
FN: This class wrongly predicted as others
TN: All correct predictions excluding that class
8. Advantages of Naïve Bayes
Fast and efficient
Works well with small datasets
Handles multi-class classification
9. Limitations
Assumes feature independence (not always true)
Performance depends on data distribution

1. Linear Regression

Linear regression is a supervised machine learning algorithm used to predict a continuous numeric value.

It finds a best-fit straight line between input (X) and output (Y).

Equation:

Y = mX + c

Meaning:

X → input (independent variable)

Y → output (dependent variable)

m → slope (change rate)

c → intercept


Example:

Predict house price based on number of rooms.

Key point:

👉 It assumes a linear relationship between variables.

📘 2. Logistic Regression

Logistic regression is used for classification problems (yes/no, 0/1).

It uses the sigmoid function to convert output into probability.

Output range:

0 to 1

Formula idea:

P = \frac{1}{1 + e^{-x}}

Example:

Spam or not spam

Purchased or not purchased


Key point:

👉 Despite the name “regression”, it is used for classification.


---

📘 3. Outliers

Outliers are data points that are very different from other observations.

Example:

Most ages = 20–40, but one value = 120 → outlier

Detection methods:

IQR method

Z-score method


IQR rule:

Below Q1 − 1.5×IQR → outlier

Above Q3 + 1.5×IQR → outlier


Key point:

👉 Outliers can distort mean and ML models.


---

📘 4. Confusion Matrix

A confusion matrix is used to evaluate classification models.

It shows correct and incorrect predictions.

Structure:

	Predicted Yes	Predicted No

Actual Yes	TP	FN
Actual No	FP	TN


Terms:

TP → True Positive

TN → True Negative

FP → False Positive

FN → False Negative


Key point:

👉 It helps measure model performance clearly.


---

📘 5. Recall

Recall tells how many actual positives were correctly predicted.

Formula:

Recall = \frac{TP}{TP + FN}

Meaning:

👉 “Out of all real positive cases, how many did we catch?”

Example:

Disease detection → important to avoid missing sick people.


---

📘 6. Precision

Precision tells how many predicted positives are actually correct.

Formula:

Precision = \frac{TP}{TP + FP}

Meaning:

👉 “Out of all predicted positives, how many are correct?”

Example:

Spam detection → avoid marking real emails as spam.


---

📘 7. Accuracy

Accuracy measures overall correctness.

Formula:

Accuracy = \frac{TP + TN}{Total}

Meaning:

👉 “How many predictions are correct overall?”

Key point:

Works well only when dataset is balanced.

📘 8. Stemming

Stemming reduces words to their root form (not always meaningful word).

Example:

playing → play

studies → studi


Method:

Simply cuts suffixes.

Key point:

👉 Fast but not always accurate.


---

📘 9. Lemmatization

Lemmatization also converts words to root form, but uses dictionary meaning.

Example:

playing → play

better → good


Key point:

👉 More accurate than stemming.


---

📘 10. TF-IDF

TF-IDF measures how important a word is in a document.

Full form:

TF → Term Frequency

IDF → Inverse Document Frequency


Formula idea:

TF = word frequency in document

IDF = importance across documents


Meaning:

👉 Common words get less weight
👉 Rare important words get higher weight

Example:

“the” → low weight

“machine learning” → high weight
